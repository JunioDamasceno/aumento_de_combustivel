Esta é uma aplicaçao do tipo Console em C#

Este programa foi criado com a finalidade de ter salvo o testedo primeiro desafio do bootcamp Localiza Labs

Desafio
Você deve calcular o consumo médio de um automóvel onde será informada a distância total percorrida (em Km) e o total de combustível consumido (em litros).

Entrada
Você receberá dois valores: um valor inteiro X com a distância total percorrida (em Km), e um valor real Y que representa o total de combustível consumido, com um dígito após o ponto decimal.

Saída
Exiba o valor que representa o consumo médio do automóvel (3 casas após a vírgula), incluindo no final a mensagem "km/l".

 
Exemplos de Entrada	e saída esperada

Distância = 500
Combustível Gasto = 35.0
Resultado esperado = 14.286 km/l

Distância = 2254
Combustível Gasto = 124.4
Resultado esperado = 18.119 km/l

Distância = 4554
Combustível Gasto = 464.6
Resultado Esperado = 9.802 km/l
